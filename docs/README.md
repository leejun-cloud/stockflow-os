# StockFlow OS 서비스 운영 매뉴얼

## 목적
이 디렉토리의 `product-plan.md`는 서비스의 기준 문서다. 제품 정의, 핵심 흐름, 데이터 모델, SaaS 가격/마케팅 가설을 계속 여기서 업데이트한다.

## 문서 운용 원칙
1. 새 기능을 만들기 전에 이 문서와 맞는지 먼저 확인한다.
2. 어댑터 규칙이 바뀌면 `product-plan.md`와 별도 어댑터 문서를 같이 갱신한다.
3. SaaS 가격/포지셔닝이 바뀌면 변경 이력 섹션을 추가한다.
