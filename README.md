# StockFlow OS

멀티 스톡사진 제출 운영 SaaS 초기 프로젝트입니다.

## 포함 내용
- `/docs/product-plan.md`: 제품 기획서 / 운영 매뉴얼 원본 Markdown
- `/src/app`: Next.js App Router 기반 웹 앱
- `/src/app/manual`: 서버에 저장된 Markdown 문서를 렌더링하는 페이지

## 실행
```bash
npm install
npm run dev
```
