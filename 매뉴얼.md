# StockFlow OS — 운영 매뉴얼

스톡 사진·영상을 여러 마켓플레이스(Adobe Stock·Shutterstock·Alamy 등)에 반자동으로 대량 업로드하는 도구. AI 메타데이터를 파일에 IPTC/XMP로 임베드하고 다중 (S)FTP로 병렬 전송한다.

## 설치
```bash
npm install
```
- `exiftool-vendored`가 exiftool 바이너리를 번들하므로 별도 시스템 설치 불필요.

## 실행
```bash
npm run dev      # 로컬 개발 (http://localhost:3000)
npm run build    # 프로덕션 빌드
npm start        # 빌드 결과 구동
npm test         # vitest
npx tsc --noEmit # 타입 체크
```

## 필수 환경변수
| 변수 | 용도 | 비고 |
|---|---|---|
| `CREDENTIALS_KEY` | 에이전시 FTP 비밀번호 AES-256-GCM 암호화 키 | **필수.** 32-byte base64. 생성: `node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"` |
| `DATABASE_URL` | Postgres 연결 문자열 | 있으면 Postgres, 없으면 로컬 SQLite. **Vercel 배포 시 필수**(파일시스템 휘발성) |
| `PGSSLMODE` | Postgres SSL 모드 | 선택 |
| `STORAGE_BACKEND` | `s3` 지정 시 S3 사용 | **Vercel 배포 시 `s3` 필수**(로컬 파일 휘발성) |
| `S3_BUCKET` / `S3_REGION` / `S3_ENDPOINT` / `S3_ACCESS_KEY_ID` / `S3_SECRET_ACCESS_KEY` / `S3_FORCE_PATH_STYLE` | S3 스토리지 설정 | MinIO 호환 |
| `COOKIE_SECURE` | 세션 쿠키 secure 플래그 | Vercel에서는 자동 |

## 배포 (Vercel)
1. `DATABASE_URL`(Postgres, 예: Neon) + S3 자격증명 + `CREDENTIALS_KEY`를 Vercel 프로젝트 환경변수에 등록.
2. DB 스키마 적용: `db/postgres-schema.sql`을 대상 Postgres에 1회 실행.
3. `git push` → Vercel 자동 배포.

> ⚠️ 로컬 기본값(SQLite + 파일시스템)은 Vercel 서버리스에서 데이터가 휘발된다. 프로덕션은 반드시 Postgres + S3.

## 자동화 범위 (합법 경계)
- **완전 자동**: AI 메타데이터 → IPTC/XMP 임베드 → 다중 (S)FTP 병렬 업로드.
- **반자동(HITL)**: 온보딩 폼 자동채움 → CAPTCHA/2FA/인증은 사용자가 처리.
- **사람 필수**: 계정 생성·W-8BEN·신분증·최종 Submit 클릭(에이전시 웹 UI).
- 1인 1계정 준수. CAPTCHA 솔버·가상번호·자동 다중계정 생성 금지(ToS 위반).

## 주요 모듈
- `src/lib/metadata-embed.ts` — IPTC/XMP 임베드(사진·영상)
- `src/lib/ftp/uploader.ts` — 다중 (S)FTP 팬아웃 + 재시도
- `src/lib/ftp/endpoints.ts` — 에이전시 FTP 엔드포인트
- `src/lib/crypto.ts` — 자격증명 암호화
- `src/lib/tax/w8ben.ts` — 한국 거주자 W-8BEN 자동 프리필(Article 14, 10%)
- `db/schema.sql` / `db/postgres-schema.sql` — 스키마
- 리서치 근거: `docs/research-blueprint-2026.md`

## 자주 쓰는 운영 팁
- 실제 업로드는 각 에이전시 실계정 + FTP 자격증명 등록 후 동작. 대시보드 "FTP Credentials"에서 등록.
- 새 에이전시 추가: `src/lib/ftp/endpoints.ts`에 엔드포인트 + `adapters/index.ts`에 메타데이터 규칙 추가.
